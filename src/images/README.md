# Images Folder Structure

This folder contains all images used in the portfolio website, organized by category.

## Structure

- `/projects` - Project-related images
  - `/web-development` - Web development project images
  - `/data-analytics` - Data analytics project images
  - `/design-creative` - Design and creative project images
  - `/robotics` - Robotics project images
- `/about` - About page images
  - `/education` - Education institution images
  - `/certificates` - Certificate images
  - `/profile` - Profile and personal images
- `/hero` - Hero section background images
- `/general` - General purpose images

## Image Naming Convention

Use descriptive, kebab-case names:
- `project-name-mockup-1.jpg`
- `certificate-aws-architect.jpg`
- `education-mit.jpg`
